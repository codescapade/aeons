class Pool<T>

package aeons.utils

@:directlyUsed

Pool is a generic object pool class that you can use to reuse objects. Objects need to be able to be instantiated one or no parameters.

Constructor

new(type:Class<T>)

Pool contructor.

Parameters:

type

The class to use in the pool.

Variables

read onlycount:Int

How many objects are currently in the pool.

Methods

clear():Void

Clear all objects out of the pool.

get(?options:Dynamic):T

Get an object from the pool. If the pool is empty create a new instance. TODO: Need to figure out how to do object pools with arguments better.

Returns:

The object.

put(object:T):Void

Put an object back into the pool. Don't keep using the object after you put it back into the pool.

Parameters:

object

The object to put back.
