interface Renderable

package aeons.core

extended by CBitmapText, CBoxShape, CLdtkTilemap, CNineSlice, CSprite, CText, CTilemap

@:directlyUsed

Interface for components that need to be rendered every frame.

Variables

anchorX:Float

The x anchor. Should be between 0 and 1.

anchorY:Float

The y anchor. Should between 0 and 1.

Methods

inCameraBounds(cameraBounds:Rect):Bool

Check if the component is inside the camera bounds and should be rendered.

Parameters:

cameraBounds

Used to render only what the camera can see. The bounds are in the local space of the component.

Returns:

True if in bounds. Out of bounds will not render using the RenderSystem.

render(target:RenderTarget):Void

Gets called every frame.

Parameters:

target

The target image to render to.
