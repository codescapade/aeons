services
Component

The component base class.

DebugRenderable

Interface to use debug drawing in components or systems.

Display

The Display manager has view and window size properties.

Entities

Entity

The Entity class is the main container class in the game. Components can be added to it for functionality.

Game

The Game class is use to start Aeons. You create a new instance of it to start it.

GameOptions

Renderable

Interface for components that need to be rendered every frame.

Scene

The Scene class is meant to be sub classes and used to initialze and run a scene in your game. Like a main menu, the game itself or a level.

SysRenderable

Interface for systems that need to be rendered every frame.

System

The system base class for ECS.

Systems

Updatable

Interface for components and systems that need to be updated when the scene updates.
