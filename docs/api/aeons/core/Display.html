interface Display

package aeons.core

The Display manager has view and window size properties.

Variables

pixelArt:Bool

When enabled everything will be rendered using nearest neighbour filtering.

read onlyviewCenterX:Int

The view center on the x axis.

read onlyviewCenterY:Int

The view center on the y axis.

read onlyviewHeight:Int

The game view height in pixels.

read onlyviewWidth:Int

The game view width in pixels.

read onlywindowCenterX:Int

The window center on the x axis.

read onlywindowCenterY:Int

The window center on the x axis.

read onlywindowHeight:Int

The window height in pixels.

read onlywindowWidth:Int

The window width in pixels.

Methods

init(designWidth:Int, designHeight:Int, pixelArt:Bool):Void

Initialize the display.

Parameters:

designWidth

The resolution width the game is designed for in pixels.

designHeight

The resolution height the game is designed for in pixels.

pixelArt

Enable optimized rendering for pixel art.

scaleToWindow():Void

Scale the view size to fit the window size. Need to add more scale modes.
