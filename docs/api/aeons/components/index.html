CBoxShape is a component to draw single color boxes.

CCamera is a component that renders a view in the render system.

CCameraOptions

The camera options you set set in the CCamera init function.

CDebugRender

CDebugRender is a component that gets added automatically to an entity that has a component that extends DebugRenderable. This goes through all renderable components and renders them.

CNapeTilemapCollider

Tilemap collider component for Nape physics.

CNineSlice
