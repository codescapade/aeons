Getting started

With Aeons you can develop on Windows, MacOS, or Linux and export to multiple platforms on desktop, web, mobile and consoles through the Kha framework.

To install Aeons have a look at the Installation section.

If you already have Aeons installed you can see how to create and use an Aeons project in Creating a new project or you can check out some of the examples.

Developing using Visual Studio Code is the preferred way, because there are extensions to help you with debugging and autocompletion. See VSCode Extensions to see the extensions to install.
